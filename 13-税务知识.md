# 税务知识

> 了解投资收益要交的税，才能真正算清楚赚了多少。

---

## 本章要点

- 股票分红持有超过1年免税
- 房产交易满5唯一最划算
- 专项附加扣除可以省不少税
- 长期持有、用好免税额度、申报专项扣除，三招省税

**现实：**
- 投资收益要交税
- 不懂税务，可能多交或少交
- 少交税 = 多赚钱

**打个比方：**

> 税务就像"过路费"——你赚钱了，国家要收一部分。懂规则的人可以合法少交，不懂的人可能多交。

**一句话记住**：懂税务不是为了逃税，而是为了合法少交。

---

## 二、常见投资收益的税率

| 投资类型 | 税率 | 说明 |
|----------|------|------|
| **股票转让** | 0% | A股暂时免征 |
| **股票分红** | 20% | 持有超过1年免税 |
| **基金分红** | 0% | 暂时免征 |
| **基金转让** | 0% | 暂时免征 |
| **债券利息** | 20% | 国债免税 |
| **银行存款利息** | 0% | 暂时免征 |
| **房产转让** | 差额的20%或全额的1%-2% | 根据情况选择 |
| **黄金转让** | 0% | 暂时免征 |

### 重点：股票分红税

**规则：**
- 持有超过1年：免税
- 持有1个月-1年：10%
- 持有不超过1个月：20%

**例子：**
> 你买了10万元股票，分红1万元：
> - 持有超过1年：税0元，到手1万元
> - 持有6个月：税1000元，到手9000元
> - 持有10天：税2000元，到手8000元

**打个比方：**

> 股票分红税就像"超市会员折扣"——持有时间越长，折扣越大。

**一句话记住**：股票分红，持有超过1年免税，这是最划算的。

## 三、房产交易税费

### 买房税费

| 税费 | 税率 | 说明 |
|------|------|------|
| **契税** | 1%-3% | 买方承担 |
| **增值税** | 5% | 卖方承担（满2年免征） |
| **个人所得税** | 差额的20%或全额的1% | 卖方承担（满5唯一免征） |
| **中介费** | 1%-3% | 买方或双方承担 |

### 卖房税费

| 情况 | 税费 |
|------|------|
| **满5唯一** | 基本免税（最划算） |
| **满2年** | 免增值税，但要交个税 |
| **不满2年** | 增值税+个税都要交 |

**打个比方：**

> 房产税费就像"结婚成本"——买房要交"彩礼"（契税），卖房要交"分手费"（增值税、个税）。持有时间越长，费用越少。

**一句话记住**：房产交易，满5唯一最划算，满2年次之。

## 四、个人所得税

### 综合所得税率

| 年应纳税所得额 | 税率 | 速算扣除数 |
|----------------|------|------------|
| 不超过36,000元 | 3% | 0 |
| 36,000-144,000元 | 10% | 2,520 |
| 144,000-300,000元 | 20% | 16,920 |
| 300,000-420,000元 | 25% | 31,920 |
| 420,000-660,000元 | 30% | 52,920 |
| 660,000-960,000元 | 35% | 85,920 |
| 超过960,000元 | 45% | 181,920 |

### 专项附加扣除

可以抵扣个税的项目：

| 项目 | 扣除标准 |
|------|----------|
| **子女教育** | 每个子女每月2000元 |
| **继续教育** | 每月400元 |
| **大病医疗** | 最高8万元/年 |
| **住房贷款利息** | 每月1000元 |
| **住房租金** | 每月800-1500元 |
| **赡养老人** | 每月3000元 |
| **婴幼儿照护** | 每个婴幼儿每月2000元 |

**打个比方：**

> 专项附加扣除就像"优惠券"——用得好可以少交不少税。

**一句话记住**：别忘了申报专项附加扣除，能省不少税。

## 五、税务优化技巧

### 1. 长期持有

- 股票分红持有超过1年免税
- 房产满5唯一免税
- 基金分红暂免征税

### 2. 合理利用免税额度

- 每人每年5万美元换汇额度
- 个人养老金每年12000元免税额度

### 3. 申报专项附加扣除

- 确保所有符合条件的项目都申报了
- 夫妻双方选择收入高的一方申报

### 4. 选择合适的投资工具

- 公募基金分红暂免征税
- 国债利息免税
- 养老金账户有税收优惠

**打个比方：**

> 税务优化就像"省钱攻略"——不是逃税，而是合法地少交税。

**一句话记住**：长期持有、用好免税额度、申报专项扣除，三招省税。
