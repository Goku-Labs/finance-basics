# 理财的正确姿势

> 理财不是买某个产品，而是一种管理钱的思维方式。

> **本章定位**：聚焦理财的**方法论和原则**——告诉你"应该怎么想"和"应该遵循什么"。关于记账实操、预算执行步骤、应急金存多少、家庭财务规划的具体操作，详见 [23-家庭财务规划](23-家庭财务规划.md)。

---

## 本章要点

- 理财 = 赚钱 + 省钱 + 存钱 + 钱生钱
- 理财三步走：先保障→再储蓄→后投资
- 六大原则：先保障、先储蓄、不懂不投、分散投、长期持、控风险
- 家庭财务报表是理财的"体检报告"
- 财富公式：财富 = 本金 × 收益率 × 时间

# 基础篇

---

## 一、理财是什么？

理财不是买某个产品，而是一种**管理钱的思维方式**。

**理财 = 赚钱 + 省钱 + 存钱 + 钱生钱**

**常见理财方式一览（风险从低到高）：**

| 方式 | 预期年化收益 | 风险 | 流动性 |
|------|------------|------|--------|
| 银行活期存款 | 0.3% | 极低 | 随时取 |
| 货币基金 | 1.5%-2.5% | 极低 | T+1到账 |
| 国债/定期存款 | 2%-3% | 极低 | 定期 |
| 债券基金 | 3%-6% | 较低 | T+1到账 |
| 混合基金 | 5%-15% | 中等 | T+1到账 |
| 股票基金 | -20%~30% | 高 | T+1到账 |
| 肧票 | -50%~100% | 很高 | T+1交易 |

> 注：收益率参考2024年市场情况，实际收益随市场波动。

**一句话记住**：理财不是暴富，而是让钱在安全的前提下，尽可能地多生一点钱。

---

## 二、预算与记账——理财的"方向盘"

### 为什么要做预算和记账？

**一句话回答**：因为大多数人"觉得自己花得不多"，实际却远超预期。

**打个比方**：
> 预算和记账就像开车时的"方向盘和仪表盘"——没有方向盘，你不知道往哪开；没有仪表盘，你不知道油还剩多少。

### 三种预算方法概览

| 方法 | 核心思路 | 适合谁 |
|------|---------|--------|
| **50/30/20法则** | 50%必要/30%想要/20%储蓄 | 大多数人 |
| **零基预算** | 每一分钱都分配去处 | 收入有限、需要精打细算的人 |
| **信封预算** | 每个类别一个"信封"，花完不补 | 容易冲动消费的人 |

> 💡 50/30/20法则的具体执行步骤和进阶版60/20/20，详见 [23-家庭财务规划](23-家庭财务规划.md)。

### 记账的核心原则

1. **分类清晰**：固定支出 vs 必要支出 vs 可选支出 vs 储蓄
2. **坚持3个月**：3个月后就能看出规律
3. **每周复盘**：不要只记不看

> 💡 记账工具推荐和实操技巧，详见 [23-家庭财务规划](23-家庭财务规划.md) 的"记账"章节。

**一句话记住**：预算是"你计划花多少"，记账是"你实际花了多少"——两者对比，才知道差距。

## 三、理财三步走

```
第一步：先保命（保险）
    ↓
第二步：存够"应急钱"（3-6个月生活费）
    ↓
第三步：让钱生钱（投资）
```

### 第一步：先保命

**配置顺序：**
1. **医保**：必须要有
2. **意外险**：第一份商业险
3. **医疗险**：补充医保
4. **重疾险**：应对大病
5. **寿险**：有家庭责任再考虑

**一句话记住**：没有保险，一场大病可能让你倾家荡产。保险是理财的地基，不是"多出来的花钱"。

### 第二步：应急金

**核心要点：**
- 应急金 = 3-6个月生活费
- 放在**随时能取**的地方（货币基金、活期存款）
- **绝对不能**放在股票或高风险基金里

**打个比方**：
> 应急金就像"灭火器"——平时觉得没用，出事时能救命。灭火器不能放在仓库最里面，要放在随手能拿到的地方。

> 💡 应急金的具体金额计算（单身/有家庭/自由职业不同标准）和存放方式对比，详见 [23-家庭财务规划](23-家庭财务规划.md)。

### 第三步：投资

**投资原则：**
- 不懂不投
- 分散投资
- 长期持有

**一句话记住**：先保命、再存底、后投资——顺序不能颠倒。

## 四、家庭财务报表——理财的"体检报告"

### 4.1 资产负债表

**定义：**
- 记录某一时点的资产和负债
- 反映家庭的财务状况

**组成：**
```
净资产 = 总资产 - 总负债
```

**例子：**

| 资产 | 金额 | 负债 | 金额 |
|------|------|------|------|
| 现金 | 5万 | 房贷 | 100万 |
| 存款 | 20万 | 车贷 | 10万 |
| 基金 | 30万 | 信用卡 | 2万 |
| 肧票 | 20万 | 其他 | 0 |
| 房产 | 200万 | | |
| **总计** | **275万** | **总计** | **112万** |

**净资产 = 275万 - 112万 = 163万**

**一句话记住**：资产负债表告诉你"你现在有多少家底"。

### 4.2 收支表

**定义：**
- 记录一段时间的收入和支出
- 反映现金流状况

**组成：**
```
结余 = 收入 - 支出
```

**例子：**

| 收入 | 金额 | 支出 | 金额 |
|------|------|------|------|
| 工资 | 15000 | 房租 | 5000 |
| 奖金 | 5000 | 餐饮 | 3000 |
| 投资收益 | 1000 | 交通 | 500 |
| 其他 | 0 | 娱乐 | 2000 |
| | | 购物 | 3000 |
| | | 储蓄 | 7500 |
| **总计** | **21000** | **总计** | **21000** |

**一句话记住**：收支表告诉你"你的钱从哪来、到哪去"。

### 4.3 财务比率分析

**常用比率：**

| 比率 | 公式 | 参考值 | 健康信号 |
|------|------|--------|----------|
| 储蓄率 | 储蓄/收入 | ≥20% | ≥20%说明你在积累财富 |
| 负债率 | 负债/总资产 | <50% | >50%说明负债过重 |
| 流动比率 | 流动资产/流动负债 | ≥2 | <2说明短期偿债能力不足 |
| 投资率 | 投资/收入 | ≥10% | ≥10%说明你在主动让钱生钱 |

**例子：**
> 月收入20000元，储蓄5000元，储蓄率=5000/20000=25%，超过20%底线，健康！

**打个比方**：
> 财务比率就像体检指标——血压、血糖、BMI，每个都有健康范围，偏离了就要注意。

**一句话记住**：储蓄率≥20%、负债率<50%，是家庭财务健康的两条红线。

## 五、六大原则

### 原则1：先保障，后投资

- 没有保险，一场大病可能让你倾家荡产
- 先配齐：医保 + 重疾险 + 意外险 + 医疗险

### 原则2：先存钱，再消费

- 工资到账，先转一部分到储蓄/投资账户
- 剩下的再花，而不是花剩下的再存

### 原则3：不懂不投

- 不懂股肧票，就不要炒股
- 不懂区块链，就不要买虚拟货币
- 只投你看得懂的东西

### 原则4：分散投资，不要把鸡蛋放一个篮子

- 不要全买股肧票
- 不要全买基金
- 肧票、债券、现金，合理搭配

### 原则5：长期持有，不要频繁操作

- 今天买明天卖，手续费都赚不回来
- 好资产，拿住，时间会给你回报

### 原则6：收益和风险成正比

- 承诺"保本高收益"的，100%是骗子
- 年化收益超过6%，就要打问号
- 年化收益超过10%，就要做好亏钱的准备

**一句话记住**：理财六大原则 = 先保障 + 先储蓄 + 不懂不投 + 分散投 + 长期持 + 控风险。

## 六、财富公式

```
财富 = 本金 × 收益率 × 时间
```

- 年轻时，**时间**是你的最大优势
- 中年时，**本金**是你的最大优势
- 任何时候，**收益率**都要建立在风险可控的基础上

**打个比方**：
> 财富就像"种树"——种子（本金）越大、土壤（收益率）越肥、时间越长，树就越高。但土壤太肥（追求高收益）可能导致根烂。

**一句话记住**：理财不是拼收益率，而是拼本金、时间和纪律。

## 常见问题

**Q：我工资很低，还需要理财吗？**
越没钱越要理财。理财不是有钱人的专利，100元也能开始定投。关键不是金额，是习惯。

**Q：理财和投资有什么区别？**
理财是"管理钱的全套方法"，包括记账、预算、保险、储蓄、投资等。投资只是理财的一部分——让钱生钱。

**Q：预算太严格，生活质量会不会下降？**
预算不是"不花钱"，而是"有计划地花钱"。50/30/20法则里30%是享受，生活质量不会下降。

**Q：六大原则里哪一条最重要？**
"不懂不投"——所有理财灾难都源于投了不懂的东西。其他原则可以慢慢修炼，这条必须立即遵守。
