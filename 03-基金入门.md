# 基金入门

> 基金是普通人最容易上手的投资工具，让专业人士帮你打理财富。

***

## 一、基金 —— 让专业人士帮你投资

**是什么？**
假设你想炒股，但完全不懂，怎么办？

基金就是：**你把钱交给一群专业的投资高手，他们帮你买股票、债券等，赚了钱大家一起分。**

**打个比方：**

> 就像你不会做菜，但想去饭店吃饭。你出钱，厨师出力，最后你吃到菜（获得收益），厨师收加工费（管理费）。

**常见类型：**

| 类型         | 买什么            | 风险 | 适合谁       |
| ---------- | -------------- | -- | --------- |
| 货币基金（如余额宝） | 短期债券、银行存单      | 极低 | 放闲钱、应急金   |
| 债券基金       | 国债、企业债         | 较低 | 求稳的钱      |
| 股票基金       | 股票             | 高  | 长期增值的钱    |
| 混合基金       | 股票+债券          | 中等 | 想平衡风险收益的钱 |
| 指数基金       | 跟踪某个指数（如沪深300） | 中等 | 最适合定投的小白  |

**一句话记住**：基金就是"凑钱请专家帮你投资"。

## 二、基金费用详解

### 2.1 管理费

**是什么？**

- 基金公司管理基金收取的费用
- 按年收取，从基金资产中扣除

**费率参考：**

| 基金类型 | 管理费率        |
| ---- | ----------- |
| 货币基金 | 0.15%-0.33% |
| 债券基金 | 0.3%-0.8%   |
| 股票基金 | 1%-1.5%     |

**例子：**

> 你买了一只管理费率1.5%的股票基金，投入1万元，每年要交150元管理费。

### 2.2 托管费

**是什么？**

- 基金托管银行收取的费用
- 用于保管基金资产、监督基金运作

**费率参考：**

- 通常0.1%-0.25%

### 2.3 销售服务费

**是什么？**

- 销售渠道（银行、支付宝等）收取的费用
- 通常按年收取

**费率参考：**

- 通常0.25%-0.8%

### 2.4 申购费/认购费

**是什么？**

- 申购费：买已经成立的基金
- 认购费：买新发行的基金

**费率参考：**

- 通常1%-1.5%
- 很多平台有折扣，甚至免费

### 2.5 赎回费

**是什么？**

- 卖出基金时收取的费用
- 持有时间越长，费率越低

**费率参考：**

| 持有时间   | 赎回费率  |
| ------ | ----- |
| 0-7天   | 1.5%  |
| 7-30天  | 0.75% |
| 30天-1年 | 0.5%  |
| 1年以上   | 0%    |

**例子：**

> 你买了1万元基金，持有6个月后赎回，赎回费率0.5%，需要交50元手续费。

### 2.6 总费用对比

**不同基金的费用结构：**

| 基金类型 | 管理费 | 托管费 | 销售服务费 | 申购费 | 赎回费 |
| ---- | --- | --- | ----- | --- | --- |
| 货币基金 | 低   | 低   | 有     | 无   | 无   |
| 债券基金 | 中   | 低   | 部分有   | 有   | 有   |
| 股票基金 | 高   | 中   | 部分有   | 有   | 有   |

**省钱技巧：**

1. 选择费率低的基金
2. 长期持有，避免频繁买卖
3. 利用平台折扣

## 三、如何挑选一只好基金

### 3.1 确定投资目标

**你的钱打算用多久？**

- **短期（1年以内）**：货币基金、短债基金
- **中期（1-3年）**：债券基金、混合基金
- **长期（3年以上）**：股票基金、指数基金

### 3.2 看基金经理

**为什么重要？**

- 主动管理型基金的业绩很大程度上取决于基金经理

**怎么看？**

1. **从业经验**：至少3年以上
2. **过往业绩**：长期业绩是否稳定
3. **管理规模**：不要太大也不要太小

### 3.3 看基金公司

**选择大公司的好处：**

- 研究实力强
- 风控体系完善
- 不容易出问题

**知名基金公司：**

- 易方达、汇添富、南方、嘉实、博时等

### 3.4 看业绩表现

**看什么？**

- **长期业绩**：3年、5年业绩
- **排名**：在同类基金中的排名
- **稳定性**：波动是否太大

**注意：**

- 不要只看短期业绩
- 过去的业绩不代表未来

### 3.5 看持仓结构

**股票基金看什么？**

- 主要持仓哪些行业
- 前十大重仓股是什么
- 是否过于集中

**债券基金看什么？**

- 主要投资什么债券
- 信用等级如何
- 久期有多长

### 3.6 看基金规模

**规模适中最好：**

- 太小：容易清盘
- 太大：灵活性差

**参考：**

- 股票基金：5亿-50亿
- 债券基金：10亿-100亿

## 四、定投策略详解

### 4.1 什么是定投？

**定义：**
每月固定日期、固定金额买入同一只基金。

**原理：**

- 跌了买得多，涨了买得少
- 长期下来拉低平均成本

**打个比方：**

> 你每月定投1000元：
>
> - 第一个月：基金净值1元，买1000份
> - 第二个月：基金净值0.8元，买1250份
> - 第三个月：基金净值1.2元，买833份
> - 平均成本：约0.98元

### 4.2 为什么定投适合小白？

**优点：**

1. **不用择时**：不管市场高低都买
2. **强制储蓄**：每月自动扣款
3. **分散风险**：长期分摊成本
4. **操作简单**：设置一次，自动执行

### 4.3 定投策略

#### 策略一：定期定额

**操作：**

- 每月固定日期买入固定金额
- 适合大多数人

#### 策略二：定期不定额

**操作：**

- 根据市场情况调整买入金额
- 市场便宜时多买，贵时少买

**参考指标：**

- 根据估值分位调整
- 根据均线调整

#### 策略三：智能定投

**操作：**

- 利用APP的智能定投功能
- 系统自动调整金额

**例子：**

- 支付宝的"慧定投"
- 天天基金的"智能定投"

### 4.4 定投金额建议

**参考公式：**

```
定投金额 = 月结余 × 投资比例
```

**投资比例参考：**

| 风险承受能力 | 投资比例    |
| ------ | ------- |
| 保守型    | 20%-30% |
| 稳健型    | 30%-50% |
| 积极型    | 50%-70% |

**例子：**

> 你每月结余3000元，风险承受能力稳健型，定投金额=3000×40%=1200元

### 4.5 定投止盈策略

**什么时候卖出？**

#### 方法一：目标收益法

- 设置一个目标收益率，达到就止盈
- 比如：收益达到50%就卖出一半

#### 方法二：估值法

- 根据估值分位止盈
- 高估值时减仓，低估值时加仓

#### 方法三：回撤法

- 当基金净值从高点回撤一定比例时止盈
- 比如：回撤20%就卖出

### 4.6 定投常见问题

**Q1：定投一定会赚钱吗？**
\*\*A1：\*\*不一定。如果市场长期下跌，定投也会亏损，但亏损幅度会比一次性买入小。

**Q2：定投多久合适？**
\*\*A2：\*\*至少3年以上，才能体现定投的优势。

**Q3：定投需要选时吗？**
\*\*A3：\*\*不需要刻意择时，但可以在估值低的时候开始。

## 五、基金交易流程

### 5.1 开户

**步骤：**

1. 选择一个平台（支付宝、天天基金、银行APP等）
2. 注册账号
3. 绑定银行卡
4. 完成风险测评

### 5.2 购买

**方式：**

- **申购**：买已经成立的基金
- **认购**：买新发行的基金

**注意：**

- 申购时间：工作日9:30-15:00
- 15:00后下单算下一个交易日

### 5.3 赎回

**步骤：**

1. 选择要赎回的基金
2. 输入赎回金额
3. 确认赎回

**到账时间：**

| 基金类型 | 到账时间    |
| ---- | ------- |
| 货币基金 | T+1     |
| 债券基金 | T+1-T+2 |
| 股票基金 | T+2-T+3 |

### 5.4 转换

**是什么？**

- 将持有的一只基金转换成另一只基金
- 节省赎回再申购的时间和费用

**注意：**

- 只能在同一家基金公司的基金之间转换
- 转换费用可能比赎回低

## 六、常见基金术语

### 6.1 净值

**是什么？**

- 基金每份的价格
- 净值=基金资产净值/总份额

**例子：**

> 一只基金资产净值1亿元，总份额1亿份，净值就是1元。

### 6.2 累计净值

**是什么？**

- 包含分红再投资的净值
- 反映基金成立以来的总收益

### 6.3 持仓

**是什么？**

- 基金持有的股票、债券等资产
- 通常每季度公布一次

### 6.4 仓位

**是什么？**

- 基金资产中股票的比例
- 股票型基金通常仓位在80%以上

### 6.5 分红

**方式：**

- **现金分红**：直接给钱
- **红利再投资**：自动买入更多份额

**选择建议：**

- 长期投资选红利再投资
- 需要用钱选现金分红
