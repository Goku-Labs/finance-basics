# 30天入门学习路线

> 系统学习金融知识，从零基础到入门。

---

## 第1周：建立认知（每天30分钟）

### Day 1：金融本质

| 学习内容 | 行动任务 |
|----------|---------|
| 读完[金融是什么](01-金融是什么.md) | 理解金融的本质是"钱的流动" |
| 了解金融市场分类 | 记住货币市场和资本市场的区别 |
| 认识主要金融机构 | 央行、商业银行、证券公司、保险公司 |

**今日收获：**
- 理解金融的核心概念
- 知道金融市场的基本结构

### Day 2：银行与储蓄

| 学习内容 | 行动任务 |
|----------|---------|
| 学习[银行与储蓄](02-银行与储蓄.md) | 打开手机银行，查看自己的账户类型 |
| 了解存款产品 | 对比活期、定期、大额存单的利率 |
| 认识银行卡种类 | 区分借记卡和信用卡 |

**今日收获：**
- 知道不同存款产品的特点
- 了解银行卡的使用技巧

### Day 3：基金入门

| 学习内容 | 行动任务 |
|----------|---------|
| 学习[基金入门](03-基金入门.md) | 在支付宝/微信查看余额宝，了解货币基金 |
| 了解基金种类 | 区分货币基金、债券基金、股票基金 |
| 认识基金费用 | 管理费、托管费、申购费、赎回费 |

**今日收获：**
- 理解基金的基本概念
- 知道不同基金的风险收益特征

### Day 4：股票入门

| 学习内容 | 行动任务 |
|----------|---------|
| 学习[股票入门](04-股票入门.md) | 下载一个股票APP，只看不动 |
| 了解股票市场 | 知道上海和深圳交易所 |
| 认识股票术语 | 开盘价、收盘价、涨跌幅 |

**今日收获：**
- 理解股票的基本概念
- 知道股票交易的基本规则

### Day 5：债券入门

| 学习内容 | 行动任务 |
|----------|---------|
| 学习[债券入门](05-债券入门.md) | 了解国债是什么，怎么买 |
| 了解债券种类 | 国债、地方债、企业债 |
| 认识债券风险 | 利率风险、信用风险 |

**今日收获：**
- 理解债券的基本概念
- 知道债券的风险类型

### Day 6：保险与保障

| 学习内容 | 行动任务 |
|----------|---------|
| 学习[保险与保障](06-保险与保障.md) | 整理自己已有的保险（医保、社保等） |
| 了解保险种类 | 意外险、医疗险、重疾险、寿险 |
| 计算保额需求 | 估算自己需要多少保额 |

**今日收获：**
- 理解保险的基本概念
- 知道如何配置保险

### Day 7：复习与总结

| 学习内容 | 行动任务 |
|----------|---------|
| 复习本周内容 | 用自己的话给家人讲一遍 |
| 做本章测试 | 检验学习效果 |
| 制定下周计划 | 确定下周的学习重点 |

**今日收获：**
- 巩固本周所学知识
- 明确下周学习目标

---

## 第2周：建立安全垫（每天30分钟）

### Day 8：应急资金

| 学习内容 | 行动任务 |
|----------|---------|
| 学习应急资金的重要性 | 计算自己3个月的生活费是多少 |
| 确定应急金金额 | 通常是3-6个月生活费 |
| 选择存放方式 | 货币基金、活期存款 |

**今日收获：**
- 知道应急金的作用
- 确定自己的应急金目标

### Day 9：开设应急账户

| 学习内容 | 行动任务 |
|----------|---------|
| 选择开户银行 | 比较不同银行的利率和服务 |
| 开设专门账户 | 单独开一个账户存应急金 |
| 设置自动转账 | 每月自动转一部分钱到应急账户 |

**今日收获：**
- 建立应急金账户
- 开始积累应急资金

### Day 10：保险配置

| 学习内容 | 行动任务 |
|----------|---------|
| 评估现有保险 | 列出自己已有的保险清单 |
| 识别保障缺口 | 看看哪些风险没有覆盖 |
| 制定保险计划 | 确定需要补充哪些保险 |

**今日收获：**
- 了解自己的保障情况
- 制定保险配置计划

### Day 11：医保与社保

| 学习内容 | 行动任务 |
|----------|---------|
| 了解医保政策 | 查询自己的医保报销比例 |
| 了解社保政策 | 查询自己的社保缴纳情况 |
| 了解公积金 | 查询公积金余额和提取条件 |

**今日收获：**
- 了解社保和医保的基本政策
- 知道如何查询个人社保信息

### Day 12：商业保险

| 学习内容 | 行动任务 |
|----------|---------|
| 对比意外险 | 比较不同保险公司的意外险产品 |
| 对比医疗险 | 比较不同保险公司的医疗险产品 |
| 对比重疾险 | 比较不同保险公司的重疾险产品 |

**今日收获：**
- 知道如何选择商业保险
- 了解不同保险产品的特点

### Day 13：制定保险计划

| 学习内容 | 行动任务 |
|----------|---------|
| 确定优先级 | 意外险→医疗险→重疾险→寿险 |
| 确定保额 | 根据家庭情况计算保额 |
| 选择产品 | 挑选适合自己的保险产品 |

**今日收获：**
- 完成保险配置计划
- 知道如何选择保险产品

### Day 14：复习与调整

| 学习内容 | 行动任务 |
|----------|---------|
| 复习本周内容 | 检查应急金账户余额 |
| 调整计划 | 根据实际情况调整目标 |
| 记录进展 | 写下本周的学习心得 |

**今日收获：**
- 巩固本周所学知识
- 调整个人财务计划

---

## 第3周：开始实践（每天30分钟）

### Day 15：基金定投入门

| 学习内容 | 行动任务 |
|----------|---------|
| 了解定投概念 | 什么是定投，为什么适合小白 |
| 学习定投策略 | 定期定额、定期不定额、智能定投 |
| 选择定投平台 | 支付宝、天天基金、银行APP |

**今日收获：**
- 理解定投的原理
- 知道如何选择定投平台

### Day 16：选择指数基金

| 学习内容 | 行动任务 |
|----------|---------|
| 了解指数类型 | 沪深300、中证500、创业板指 |
| 比较指数基金 | 选择费用低、跟踪误差小的基金 |
| 查看基金业绩 | 看3年、5年业绩排名 |

**今日收获：**
- 知道如何选择指数基金
- 了解不同指数的特点

### Day 17：开设基金账户

| 学习内容 | 行动任务 |
|----------|---------|
| 选择平台开户 | 在支付宝/天天基金/银行APP开户 |
| 完成风险测评 | 确定自己的风险承受能力 |
| 绑定银行卡 | 绑定常用银行卡 |

**今日收获：**
- 完成基金账户开户
- 知道如何进行风险测评

### Day 18：第一次定投

| 学习内容 | 行动任务 |
|----------|---------|
| 选择基金 | 挑选一只指数基金 |
| 设置定投金额 | 每月自动定投（金额自定） |
| 设置定投日期 | 选择发工资后的日期 |

**今日收获：**
- 完成第一次定投设置
- 开始实践基金投资

### Day 19：资产配置入门

| 学习内容 | 行动任务 |
|----------|---------|
| 了解资产配置 | "股债平衡"的概念 |
| 确定配置比例 | 根据风险承受能力确定比例 |
| 构建投资组合 | 选择不同类型的资产 |

**今日收获：**
- 理解资产配置的重要性
- 知道如何构建投资组合

### Day 20：制定个人配置

| 学习内容 | 行动任务 |
|----------|---------|
| 确定配置比例 | 如：股票基金50%+债券基金30%+货币基金20% |
| 选择具体产品 | 挑选符合配置比例的基金 |
| 记录配置方案 | 写下自己的资产配置计划 |

**今日收获：**
- 完成个人资产配置计划
- 知道如何选择投资产品

### Day 21：复习与记录

| 学习内容 | 行动任务 |
|----------|---------|
| 建立投资笔记 | 记录自己的操作和心得 |
| 回顾投资组合 | 检查配置比例是否合理 |
| 设定观察期 | 决定多久检查一次投资组合 |

**今日收获：**
- 建立投资记录习惯
- 明确投资组合管理计划

---

## 第4周：进阶与巩固（每天30分钟）

### Day 22：财报基础

| 学习内容 | 行动任务 |
|----------|---------|
| 了解利润表 | 收入、成本、利润 |
| 了解资产负债表 | 资产、负债、净资产 |
| 了解现金流量表 | 经营、投资、筹资现金流 |

**今日收获：**
- 知道三大财务报表的基本结构
- 理解财务分析的基础

### Day 23：宏观经济

| 学习内容 | 行动任务 |
|----------|---------|
| 了解GDP | 国内生产总值的概念 |
| 了解CPI | 消费者价格指数的作用 |
| 关注利率政策 | 央行的货币政策 |

**今日收获：**
- 了解宏观经济指标
- 知道宏观经济对投资的影响

### Day 24：投资心理

| 学习内容 | 行动任务 |
|----------|---------|
| 了解常见心理偏差 | 追涨杀跌、锚定效应、损失厌恶 |
| 学习克服方法 | 如何保持理性投资 |
| 制定投资纪律 | 设定买入和卖出规则 |

**今日收获：**
- 认识投资中的心理误区
- 知道如何保持理性

### Day 25：复盘投资行为

| 学习内容 | 行动任务 |
|----------|---------|
| 回顾投资操作 | 检查是否有冲动消费/投资 |
| 分析决策过程 | 哪些决策是理性的，哪些是情绪化的 |
| 制定改进计划 | 如何避免情绪化决策 |

**今日收获：**
- 了解自己的投资行为模式
- 制定改进计划

### Day 26：税务知识

| 学习内容 | 行动任务 |
|----------|---------|
| 了解个人所得税 | 税率、扣除项 |
| 了解资本利得税 | 投资收益的税收政策 |
| 了解税收优惠 | 哪些投资可以免税 |

**今日收获：**
- 了解基本税务知识
- 知道如何合法节税

### Day 27：制定长期计划

| 学习内容 | 行动任务 |
|----------|---------|
| 设定财务目标 | 5年/10年的财务目标 |
| 计算所需金额 | 需要多少钱才能实现目标 |
| 制定行动计划 | 每年需要存多少钱、投资多少 |

**今日收获：**
- 明确长期财务目标
- 制定实现计划

### Day 28：建立记账习惯

| 学习内容 | 行动任务 |
|----------|---------|
| 选择记账工具 | 下载记账APP |
| 设定记账分类 | 餐饮、交通、娱乐等 |
| 开始记录 | 记录每笔收支 |

**今日收获：**
- 建立记账习惯
- 开始跟踪个人收支

### Day 29：阅读一本理财书

**推荐书籍：**
1. 《小狗钱钱》—— 理财启蒙，故事有趣
2. 《富爸爸穷爸爸》—— 建立财商思维
3. 《指数基金投资指南》—— 小白投资实操
4. 《聪明的投资者》—— 价值投资经典

**今日任务：**
- 选择一本开始阅读
- 做读书笔记

**今日收获：**
- 学习更多理财知识
- 建立阅读习惯

### Day 30：总结与展望

| 学习内容 | 行动任务 |
|----------|---------|
| 回顾30天收获 | 写下学习心得 |
| 评估学习效果 | 检查是否达到预期目标 |
| 制定下阶段计划 | 确定下一步学习方向 |

**今日收获：**
- 总结30天学习成果
- 规划未来学习方向

## 最重要的三句话

1. **理财不是一夜暴富，而是慢慢变富**
2. **不懂的东西不要碰，保住本金是第一位的**
3. **最好的投资时间是十年前，其次是现在**
