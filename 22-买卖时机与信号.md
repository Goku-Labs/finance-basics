# 22-买卖时机与信号

> 写给完全零基础的你：学会看指标、看估值、设止盈，投资就不再"凭感觉"。

---

## 本章要点

- 技术指标（均线、MACD、RSI）帮你看趋势、找信号
- 估值指标（PE百分位、股息率）帮你看贵不贵
- 止盈止损策略帮你保住利润、控制亏损
- 没有100%准确的指标，组合使用才靠谱

# 基础篇

---

## 一、K线——看懂最基础的"价格图"

### 什么是K线？

K线就是记录一天价格变化的"小蜡烛"，包含4个价格：

| 名称 | 含义 |
|------|------|
| **开盘价** | 当天第一笔成交价 |
| **收盘价** | 当天最后一笔成交价 |
| **最高价** | 当天最高的成交价 |
| **最低价** | 当天最低的成交价 |

### 阳线和阴线

- **阳线**（红色/空心）：收盘价 > 开盘价，当天涨了
- **阴线**（绿色/实心）：收盘价 < 开盘价，当天跌了

> 打个比方：K线就像"一天的战斗记录"——红色代表多头（买方）赢了，绿色代表空头（卖方）赢了。

### 上下影线

- **上影线**：最高价到实体的线，代表上方有抛压
- **下影线**：最低价到实体的线，代表下方有支撑

**举例：**
一根长下影线的阳线 → 当天跌下去又被拉上来 → 说明有人在下方接盘

**一句话记住**：阳线看涨、阴线看跌，影线代表"试探过但没守住"。

---

## 二、均线（MA）——看趋势的"平滑线"

### 什么是均线？

均线就是把过去N天的收盘价平均一下，连成一条线。它能帮你看出价格的"大方向"。

> 打个比方：均线就像"人群的平均身高"——单个人忽高忽低看不出规律，但平均一下就能看出整体趋势。

### 常见均线

| 均线 | 周期 | 代表含义 |
|------|------|----------|
| **5日均线** | 最近5天 | 短期趋势（一周） |
| **10日均线** | 最近10天 | 短中期趋势（两周） |
| **20日均线** | 最近20天 | 中期趋势（一个月） |
| **60日均线** | 最近60天 | 中长期趋势（一个季度） |
| **250日均线** | 最近250天 | 长期趋势（一年） |

### 金叉和死叉

**金叉（买入信号）：**
- 短期均线从下往上穿过长期均线
- 比如5日均线向上穿过20日均线
- 说明短期趋势变强，可能要涨

**死叉（卖出信号）：**
- 短期均线从上往下穿过长期均线
- 比如5日均线向下穿过20日均线
- 说明短期趋势变弱，可能要跌

> 打个比方：金叉就像"短跑选手追上了长跑选手"——说明最近跑得快，势头正猛；死叉就像"短跑选手被长跑选手甩开了"——说明最近跑不动了。

### 均线的支撑和压力

- 股价在均线上方 → 均线是"支撑线"（跌到均线附近可能反弹）
- 股价在均线下方 → 均线是"压力线"（涨到均线附近可能回落）

**一句话记住**：金叉看涨、死叉看跌，均线是趋势的"导航仪"。

---

## 三、MACD——看动量的"温度计"

### 什么是MACD？

MACD由三条线组成：

| 线 | 名称 | 含义 |
|----|------|------|
| **DIFF** | 快线 | 短期均线与长期均线的差值 |
| **DEA** | 慢线 | DIFF的平均值 |
| **柱状图** | MACD柱 | DIFF与DEA的差值 |

### 怎么看MACD？

**金叉（买入信号）：**
- DIFF从下往上穿过DEA
- 柱状图由绿变红
- 说明动量转正，可能要涨

**死叉（卖出信号）：**
- DIFF从上往下穿过DEA
- 柱状图由红变绿
- 说明动量转负，可能要跌

**零轴上方/下方：**
- MACD在零轴上方 → 整体趋势偏多
- MACD在零轴下方 → 整体趋势偏空

### MACD背离

**顶背离（卖出信号）：**
- 股价创新高，但MACD没有创新高
- 说明上涨动力不足，可能要跌

**底背离（买入信号）：**
- 股价创新低，但MACD没有创新低
- 说明下跌动力不足，可能要涨

> 打个比方：MACD就像"汽车的速度表"——金叉是踩油门加速，死叉是踩刹车减速，背离是"油门踩了但速度没上去"，说明发动机有问题了。

**一句话记住**：MACD金叉看涨、死叉看跌，背离是重要的转折信号。

---

## 四、RSI——看超买超卖的"体温计"

### 什么是RSI？

RSI（相对强弱指标）衡量最近一段时间内涨跌的力量对比，数值在0-100之间。

### 怎么看RSI？

| RSI数值 | 含义 | 操作建议 |
|---------|------|----------|
| **>70** | 超买区 | 涨太多了，可能要回调，考虑卖出 |
| **50附近** | 中性区 | 方向不明，观望 |
| **<30** | 超卖区 | 跌太多了，可能要反弹，考虑买入 |

> 打个比方：RSI就像"人的体温"——超过37.5度是发烧（超买），低于36度是低体温（超卖），正常范围是36-37.5度。

### RSI的注意事项

- RSI在强势市场可能长期处于超买区（>70），不要一超卖就卖
- RSI在弱势市场可能长期处于超卖区（<30），不要一超卖就买
- 结合其他指标一起看更靠谱

**一句话记住**：RSI>70是"涨太多了"，RSI<30是"跌太多了"。

## 五、布林带——看波动的"通道"

### 什么是布林带？

布林带由三条线组成：

| 线 | 含义 |
|----|------|
| **上轨** | 中轨 + 2倍标准差 |
| **中轨** | 20日均线 |
| **下轨** | 中轨 - 2倍标准差 |

### 怎么看布林带？

**突破上轨：**
- 股价涨到上轨以上 → 可能超买，注意回调
- 但也可能是强势上涨的开始

**跌破下轨：**
- 股价跌到下轨以下 → 可能超卖，注意反弹
- 但也可能是继续下跌的信号

**收口（变盘信号）：**
- 布林带上下轨距离缩小 → 波动率降低 → 可能要变盘（大涨或大跌）

**开口（趋势确认）：**
- 布林带上下轨距离扩大 → 波动率增加 → 趋势可能延续

> 打个比方：布林带就像"河道"——股价在河道里跑，跑到岸边（上轨/下轨）可能被弹回来，河道变窄（收口）可能要发洪水（变盘）。

**一句话记住**：布林带是"价格的通道"，收口是变盘信号，突破看趋势。

## 六、估值买入法——看"贵不贵"

技术指标看趋势，估值指标看贵不贵。两者结合才靠谱。

### 1. PE百分位——最常用的估值指标

**什么是PE？**
PE = 股价 ÷ 每股利润，代表"回本需要多少年"。

**什么是PE百分位？**
当前PE在历史数据中的位置。比如PE百分位30%，意思是历史上70%的时间PE比现在高。

| PE百分位 | 含义 | 操作建议 |
|----------|------|----------|
| **<30%** | 低估区 | 便宜，适合买入 |
| **30%-70%** | 合理区 | 正常，可以持有 |
| **>70%** | 高估区 | 贵了，考虑卖出 |

> 打个比方：PE百分位就像"打折力度"——30%百分位相当于"7折促销"，70%百分位相当于"涨价了"。

**怎么查？**
- 且慢、蛋卷基金等APP可以查指数的PE百分位
- 理杏仁、乌龟量化等网站也可以查

### 2. PB（市净率）

**什么是PB？**
PB = 股价 ÷ 每股净资产，代表"你花多少钱买1元的净资产"。

**适用场景：**
- 银行、地产等重资产行业用PB更合适
- PB < 1 表示"破净"，可能被低估

### 3. 股息率

**什么是股息率？**
股息率 = 每股分红 ÷ 股价，代表"每年能拿到多少分红"。

| 股息率 | 含义 |
|--------|------|
| **>4%** | 高股息，适合长期持有 |
| **2%-4%** | 正常 |
| **<2%** | 低股息，不太适合分红策略 |

**一句话记住**：PE百分位<30%是"打折"，>70%是"涨价"——打折时买，涨价时卖。

## 七、止盈止损策略——保住利润、控制亏损

### 1. 目标收益率止盈

**方法：** 设定一个目标收益率（如20%或30%），达到就卖出。

**举例：**
- 10元买入，目标收益率20%，涨到12元就卖
- 优点：简单明确
- 缺点：可能卖早了，错过后续上涨

### 2. 回撤止盈

**方法：** 从最高点回撤一定比例（如10%）就卖出。

**举例：**
- 10元买入，涨到15元（最高点）
- 从15元回撤10%，跌到13.5元就卖
- 优点：能吃到大部分上涨
- 缺点：回撤幅度设太小容易被洗出去

### 3. 移动止盈（最推荐）

**方法：** 随着利润增加，不断提高止盈线。

**举例：**
- 10元买入
- 涨到12元（利润20%），止盈线设11元（回撤50%利润）
- 涨到15元（利润50%），止盈线设13.5元（回撤30%利润）
- 涨到18元（利润80%），止盈线设16元（回撤约20%利润）

> 打个比方：移动止盈就像"下山的绳子"——爬得越高，绳子越长，即使摔一跤也不会掉到山底。

### 4. 网格交易

**方法：** 设定一个价格区间，每跌X%买一份，每涨X%卖一份。

**举例：**
- 股价在10-15元之间波动
- 每跌5%买一份，每涨5%卖一份
- 10元买一份 → 9.5元买一份 → 10元卖一份 → 10.5元卖一份

**优点：** 不需要预测方向，适合震荡市
**缺点：** 单边上涨或下跌时效果不好

### 5. 定投止盈

**方法：** 定投过程中，当估值达到高估区域时卖出。

**举例：**
- 每月定投沪深300ETF
- PE百分位 < 30%：正常定投
- PE百分位 30%-70%：减少定投
- PE百分位 > 70%：开始分批卖出

**一句话记住**：止盈是"落袋为安"，止损是"断臂求生"——两者都要提前设好。

## 八、综合实战——买入和卖出检查清单

### 买入检查清单

| 检查项 | 要求 |
|--------|------|
| 趋势 | 股价在20日均线上方（趋势向上） |
| 估值 | PE百分位 < 50%（不贵） |
| 信号 | MACD金叉或RSI < 40（有买入信号） |
| 仓位 | 单只股票不超过总资产20% |
| 资金 | 用闲钱，不用生活费 |

### 卖出检查清单

| 检查项 | 触发条件 |
|--------|----------|
| 止盈 | 达到目标收益率或移动止盈线 |
| 止损 | 跌破止损线（如买入价-10%） |
| 估值 | PE百分位 > 70%（高估） |
| 信号 | MACD死叉或RSI > 70（有卖出信号） |
| 基本面 | 公司业绩大幅下滑或出现重大利空 |

## 九、常见错误与纠正

| 错误 | 表现 | 纠正 |
|------|------|------|
| 只看一个指标 | 只看MACD金叉就买 | 至少看2-3个指标确认 |
| 频繁交易 | 每天看盘买卖 | 减少交易频率，按计划操作 |
| 不设止损 | 跌了就死扛 | 提前设好止损线 |
| 贪心不卖 | 涨了还想涨更多 | 用移动止盈锁定利润 |
| 追涨杀跌 | 涨了追买、跌了割肉 | 按估值和信号操作 |
