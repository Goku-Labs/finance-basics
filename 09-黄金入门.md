# 黄金入门

> 黄金是人类历史上最古老的"硬通货"，也是重要的避险资产。

***

## 一、黄金 —— 永不褪色的"硬通货"

**是什么？**

黄金是一种**天然的贵金属**，具有稀有、耐腐蚀、易分割等特性，几千年来一直被当作货币和财富的象征。

**打个比方：**

> 想象你有一张"财富保险单"，无论世界发生什么变化，这张保单永远有效，可以兑换成任何国家的货币。黄金就是这样一张"保单"。

**为什么重要？**

| 原因 | 说明 |
|------|------|
| **稀缺性** | 地球上的黄金总量有限，开采成本高 |
| **通用性** | 全球公认的价值储存手段 |
| **避险属性** | 经济不稳定时，人们会买黄金保值 |
| **抗通胀** | 长期来看能抵御通货膨胀 |

**一句话记住**：黄金是"硬通货"，是财富的"压舱石"。

***

## 二、投资黄金的方式

### 2.1 实物黄金

**常见形式：**

| 形式 | 说明 | 适合谁 |
|------|------|--------|
| **金条** | 纯度高，价格接近国际金价 | 长期保值、送礼 |
| **金币** | 有收藏价值，价格可能高于金价 | 收藏爱好者 |
| **金饰** | 美观但溢价高，变现难 | 佩戴装饰 |

**优点**：看得见摸得着，心里踏实

**缺点**：需要保管，有被盗风险，交易成本高

### 2.2 纸黄金

**是什么？**

银行提供的虚拟黄金交易，不涉及实物交割。

**优点**：
- 不用保管实物
- 交易方便，成本低
- 可以小金额投资

**缺点**：
- 只是账户数字，没有实物
- 银行可能收取管理费

### 2.3 黄金ETF

**是什么？**

交易所交易基金，追踪黄金价格。

**优点**：
- 像股票一样方便交易
- 费用低
- 流动性好

**缺点**：
- 需要证券账户
- 有管理费用

### 2.4 黄金基金

**是什么？**

基金公司发行的投资黄金的基金。

**优点**：
- 门槛低，1元起投
- 专业管理
- 可以定投

**缺点**：
- 有管理费用
- 净值波动

**一句话记住**：想方便选ETF或纸黄金，想踏实选实物黄金。

## 三、黄金什么时候涨？

### 3.1 利好因素

| 因素 | 说明 |
|------|------|
| **经济不稳定** | 战争、疫情、金融危机时 |
| **通货膨胀** | 物价上涨，钱不值钱时 |
| **美元走弱** | 黄金和美元通常反向变动 |
| **央行购金** | 各国央行增加黄金储备 |

### 3.2 利空因素

| 因素 | 说明 |
|------|------|
| **经济强劲** | 股市好，资金流向股票 |
| **利率上升** | 持有黄金没有利息，机会成本高 |
| **美元走强** | 黄金以美元计价，美元贵了黄金就显得贵 |

**一句话记住**：乱世买黄金，盛世买股票。

***

## 四、投资黄金的注意事项

### 4.1 黄金不是稳赚不赔

**风险提醒：**

- 黄金也会跌！2011年到2015年，金价跌了近40%
- 短期波动大，不适合短期投机
- 没有利息收入

### 4.2 配置比例

**建议**：黄金在总资产中的占比建议在5%-10%。

> 💡 **为什么？** 黄金主要是用来避险和分散风险的，不是用来赚钱的。配置太多会影响整体收益。

### 4.3 哪里买？

| 渠道 | 说明 |
|------|------|
| **银行** | 买实物黄金、纸黄金 |
| **证券公司** | 买黄金ETF |
| **基金平台** | 买黄金基金 |
| **正规金店** | 买金饰、金条 |

**一句话记住**：黄金是"保险"不是"股票"，别指望它暴富。

## 五、总结

| 要点 | 内容 |
|------|------|
| 是什么 | 天然贵金属，硬通货 |

**一句话记住**：黄金是财富的"压舱石"，不是用来投机的工具。
