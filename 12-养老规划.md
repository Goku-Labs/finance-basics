# 养老规划

> 越早规划养老，退休生活越从容。

---

## 本章要点

- 社保养老金只能保基本，需要自己补充
- 养老三支柱：社保+企业年金+个人养老金
- 越早开始，复利效应越大，压力越小
- 养老投资要"越老越稳"，不要冒险

### 养老金缺口

**现实：**
- 社保养老金只能覆盖基本生活
- 想要体面退休，需要自己补充

**打个比方：**

> 社保养老金就像"米饭"——能吃饱，但想吃肉得自己买。

**养老金计算：**
```
养老金缺口 = 期望退休生活费 × 预期寿命 - 社保养老金
```

**例子：**
> 你希望退休后每月花5000元，活到85岁，社保每月给3000元：
> - 缺口：(5000-3000) × 12个月 × 25年 = 60万元
> - 这60万需要自己准备

**一句话记住**：社保是基础，自己补充是关键。

## 二、养老三支柱

| 支柱 | 说明 | 特点 |
|------|------|------|
| **第一支柱** | 社保养老金 | 国家给的，保基本 |
| **第二支柱** | 企业年金/职业年金 | 单位给的，不是所有公司都有 |
| **第三支柱** | 个人养老金 | 自己存的，最可控 |

### 第一支柱：社保养老金

**怎么交？**
- 单位交16%，个人交8%
- 交满15年，退休后按月领

**能领多少？**
- 和缴费年限、缴费基数、当地平均工资有关
- 一般来说，替代率约40%-60%（退休后收入/退休前收入）

### 第二支柱：企业年金

**是什么？**
企业为员工建立的补充养老保险。

**特点：**
- 企业出钱，个人也出钱
- 不是所有公司都有
- 退休后按月领取

### 第三支柱：个人养老金

**是什么？**
自己开立账户，每年最多存12000元，享受税收优惠。

**优点：**
- 税收优惠（最高可省5400元税/年）
- 强制储蓄
- 专款专用

**怎么开户？**
- 银行APP
- 支付宝/微信
- 具体产品：养老目标基金、养老保险等

**打个比方：**

> 养老三支柱就像"三条腿的凳子"——社保是第一条腿，企业年金是第二条腿，个人养老金是第三条腿。腿越多越稳。

**一句话记住**：三条腿走路，养老才稳当。

## 三、养老投资策略

### 核心原则

| 原则 | 说明 |
|------|------|
| **早开始** | 时间是最好的朋友 |
| **稳为主** | 临近退休，风险要降低 |
| **多元化** | 不要把所有钱放一个篮子 |
| **定期调整** | 每年检查一次，根据情况调整 |

### 不同年龄段策略

| 年龄段 | 风险偏好 | 投资建议 |
|--------|----------|----------|
| **25-35岁** | 积极 | 股票基金为主，追求高增长 |
| **35-45岁** | 稳健 | 股债平衡，逐步降低风险 |
| **45-55岁** | 保守 | 债券为主，保本为先 |
| **55岁以上** | 极保守 | 存款、国债，确保安全 |

### 养老目标基金

**是什么？**
专门为养老设计的基金，根据退休日期自动调整资产配置。

**特点：**
- 目标日期越近，股票比例越低
- 一站式解决方案
- 省心省力

**例子：**
> "养老目标2050基金"——适合2050年左右退休的人，现在股票比例高，随着时间推移逐渐降低。

**打个比方：**

> 养老投资就像"马拉松"——年轻时跑快点（高风险），中年时保持节奏（平衡），老年时减速慢行（低风险）。

**一句话记住**：养老投资要"越老越稳"，不要冒险。

## 四、养老规划步骤

### 第一步：估算退休后需要多少钱

**考虑因素：**
- 退休后生活费（现在的70%-80%）
- 医疗费用（会增加）
- 通货膨胀（钱会贬值）
- 预期寿命

### 第二步：计算养老金缺口

**公式：**
```
缺口 = 退休后总支出 - 社保养老金 - 企业年金
```

### 第三步：制定投资计划

**根据缺口计算每月需要存多少：**
```
每月存款 = 缺口 ÷ (1+收益率)^年数 ÷ 12
```

### 第四步：执行并定期检查

**每年检查一次：**
- 收入变化
- 投资收益
- 生活成本变化
- 调整计划

**打个比方：**

> 养老规划就像"存钱买房"——先算总价（需要多少钱），再算首付（社保+企业年金），剩下的分期付款（每月定投）。

**一句话记住**：养老规划四步走：估算需求→计算缺口→制定计划→执行检查。

## 五、常见误区

### 误区1：我还年轻，不用规划养老

**真相：** 越早开始，复利效应越大，压力越小。

> 25岁开始每月存1000元，60岁能有约100万（按年化7%计算）
> 35岁开始每月存1000元，60岁只有约45万

### 误区2：社保够用了

**真相：** 社保替代率约40%-60%，退休后收入大幅下降，需要自己补充。

### 误区3：退休后再规划

**真相：** 退休后再规划已经晚了，时间不够，压力太大。

### 误区4：养老就是存钱

**真相：** 存钱会贬值，需要投资跑赢通胀。

**打个比方：**

> 这些误区就像"不系安全带开车"——平时觉得没事，出事时后悔莫及。

**一句话记住**：养老要趁早，别等退休才后悔。
