# 保险与保障

> 保险是风险管理的工具，用小钱防范大风险。

---

## 本章要点

- 保险是"众筹防风险"，平时花小钱，关键时刻救命
- 配置顺序：医保→意外险→医疗险→重疾险→寿险
- 三个原则：先保障后理财、先大人后小孩、如实告知
- 保额不是越高越好，够用就行

---

# 基础篇

---

## 一、保险 —— 花小钱，防大灾

**是什么？**
保险就是**"风险转移"**的工具。

**打个比方：**
> 你花200元买了一份意外险。万一出意外，保险公司赔你50万。如果没出意外，这200元就"消费"掉了。

**核心逻辑：**
- 一个人出事概率低，但万一出事损失巨大
- 1000个人每人交一点钱，凑成一个大池子
- 谁出事，就从池子里拿钱赔给他

**一句话记住**：保险是"众筹防风险"，平时花小钱，关键时刻救命。

---

## 二、核心保险类型

| 类型 | 作用 | 一句话理解 |
|------|------|------------|
| **医保（社保）** | 基本看病报销 | 国家给的"基础保障" |
| **意外险** | 意外受伤/身故赔钱 | "防倒霉"的保险 |
| **医疗险** | 看病报销（补充医保） | "报销型"，花多少报多少 |
| **重疾险** | 得大病，一次性赔一笔钱 | "给付型"，确诊就给钱 |
| **寿险** | 人不在了，给家人留一笔钱 | "留爱不留债" |

### 重疾险 vs 医疗险

| 对比项 | 重疾险 | 医疗险 |
|--------|--------|--------|
| 赔付方式 | 确诊即赔一笔钱 | 实报实销 |
| 用途 | 弥补收入损失、康复费用 | 支付医疗费用 |
| 价格 | 较贵（几千到上万/年） | 较便宜（几百/年） |

**打个比方：**
> 重疾险像"失业金"——确诊大病后给你一笔钱，让你安心养病不用上班；医疗险像"报销单"——看病花了多少，保险公司给你报多少。

**一句话记住**：重疾险管"收入损失"，医疗险管"医疗费用"，两个都要买。

---

## 三、配置优先级

**新手买保险，按这个顺序来：**

| 优先级 | 险种 | 原因 |
|--------|------|------|
| ★★★★★ | 医保 | 国家给的，必须要有 |
| ★★★★★ | 意外险 | 便宜，保障高，人人需要 |
| ★★★★☆ | 医疗险 | 补充医保报销缺口 |
| ★★★★☆ | 重疾险 | 防大病导致家庭陷入困境 |
| ★★★☆☆ | 寿险 | 有家庭责任再考虑 |

### 三个核心原则

1. **先保障后理财**：先买保障型保险，再考虑理财型保险
2. **先大人后小孩**：先给家庭经济支柱买，再给孩子买
3. **如实告知**：健康状况要如实说，否则可能拒赔

**打个比方：**
> 买保险就像"穿雨衣"——先穿好基础款（医保+意外险+医疗险），再考虑高级款（重疾险+寿险）。

**一句话记住**：医保+意外险+医疗险是基础，重疾险+寿险是升级。

---

## 四、常见误区

| 误区 | 真相 |
|------|------|
| **保险是骗人的** | 保险是合法的金融产品，受监管机构监管，理赔有合同保障 |
| **有医保就够了** | 医保报销比例有限，有很多费用不报（如进口药、自费药），需要商业保险补充 |
| **买保险不划算** | 保险是防范风险的工具，不出事最好，出事了能救命。不能用"划算"来衡量 |
| **理财型保险收益高** | 理财型保险收益通常低于基金，主要功能是保障，不是理财。新手不建议买 |

**打个比方：**
> 这些误区就像"不用系安全带"——平时觉得没用，出事时后悔莫及。

**一句话记住**：保险不是消费，是保护；不是投资，是保障。

## 五、保额怎么算

### 重疾险保额

**计算公式：**
```
重疾险保额 = 治疗费用 + 康复费用 + 家庭开支 × 3-5年
```

**参考：**
- 治疗费用：30-50万元
- 康复费用：10-20万元
- 家庭开支：根据实际情况

**例子：**
> 你年收入20万元，重疾险保额至少需要：30万（治疗）+ 10万（康复）+ 20万×3年 = 100万元

### 寿险保额

**计算公式：**
```
寿险保额 = 房贷 + 车贷 + 子女教育费用 + 家庭开支 × 5-10年
```

**例子：**
> 你有房贷100万，车贷20万，子女教育需要50万，家庭每年开支10万，寿险保额至少需要：100+20+50+10×10=270万元

### 医疗险保额

**建议：**
- 至少100万元
- 最好包含住院前后门急诊、门诊手术等

### 意外险保额

**建议：**
- 至少50万元
- 经常出差的人可以买更高

**打个比方：**
> 保额就像"安全网的大小"——网太小接不住，网太大浪费钱。要根据家庭实际情况来定。

**一句话记住**：保额不是越高越好，够用就行。

## 六、理赔流程

### 报案

**时间：** 出险后10日内

**方式：**
- 拨打保险公司客服电话
- 通过APP报案
- 联系保险代理人

### 提交材料

| 险种 | 需要材料 |
|------|---------|
| 医疗险 | 医疗费用发票、诊断证明、费用清单 |
| 重疾险 | 诊断证明、病理报告 |
| 意外险 | 意外事故证明、医疗费用发票 |
| 寿险 | 死亡证明、户籍注销证明 |

### 审核与赔付

- 审核时间：一般30日内
- 赔付方式：直接转账到指定账户

### 理赔注意事项

1. **及时报案**：不要拖延
2. **保留证据**：保留所有相关票据
3. **如实告知**：不要隐瞒病情

**打个比方：**
> 理赔就像"报销差旅费"——准备好发票（材料），交给财务（保险公司），审核通过就给你打钱。

**一句话记住**：材料齐全、如实告知、及时报案，理赔不难。

## 七、购买技巧

### 对比多家产品

**看什么？**
- 保障范围（越全越好）
- 保费（同等保障选便宜的）
- 理赔服务（看口碑）

### 选择合适的期限

| 险种 | 建议期限 |
|------|----------|
| 重疾险 | 保到70岁或终身 |
| 寿险 | 保到60岁或65岁 |
| 医疗险 | 每年续保 |
| 意外险 | 每年续保 |

### 注意细节

- **等待期**：重疾险通常有90-180天等待期，等待期内发病不赔
- **免赔额**：医疗险通常有1万元免赔额，1万元以下不赔
- **免责条款**：看清哪些情况不赔

**打个比方：**
> 买保险就像"买衣服"——要看款式（保障范围）、价格（保费）、尺码（保额），还要看售后（理赔服务）。

**一句话记住**：货比三家，看清条款，不盲目跟风。

## 常见问题

**Q：重疾险和医疗险有什么区别？**
重疾险是确诊大病就赔一笔钱，可以自由支配；医疗险是看病花了多少报多少，只能报销医疗费用。

**Q：保险理赔难吗？**
只要如实告知、材料齐全、及时报案，理赔并不难。大多数拒赔是因为没有如实告知或不符合赔付条件。
